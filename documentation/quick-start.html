key Note:-

Global - School Multi management system express is basically support for manage any kind of education based organization like school, college, universities etc.

GMSMS Installation:-

1. System Requirements

*** PHP 5.3+

*** MYSQL 5+

*** PHP Mcrypt Extention

*** PHP MBString Extension

*** PHP DOM Extension

*** PHP GD Extension

*** PHP MYSQL Extension

*** PHP PDO Extention

*** PHP CURL Extention

*** Internet Connection

2. Installation

  1. Note: if you have ssl certificate for your domain, then please edit the following file application/config/config.php at 26th line with https instead of http
  2. File For Localhost: Place the downloaded zip file to your local server xampp htdocs ( or your own folder inside htdocs) directory and extract the uploaded zip file
  3. File For Live: Upload the downloaded zip file to your hosting server /public_html/ directory. After complete upload then extract the uploaded zip file
  4. Each requirements should be 100%, not try to skip any server requirements. Otherwise software will not run smoothly
  5. Please input valid purchase code. Otherwise software will not run properly.
  6. Databse For Localhost: Manually create a database from your localhost phpMyAdmin and Import database.sql file from documentation/database
  7. Database For Live: Manually create a database from your server cpanel using MySql Database Wizard and Create a user for the database and link that user to the created database with all necessary permissions and Import database.sql file from documentation/database
  8. Run the project from any web browser like (http://www.yourdomain.com/).
  9. Now Please follow each steps very well and accurately.

Super Admin Have To Do:-

After login Super Admin must have to do following things

  1. General/Global Setting from Left Menu->Administrator->General Setting
  2. Create School from Left Menu->Administrator->Manage School
  3. Set user Role/Type wise access permission from Left Menu->Administrator->Role Permission(ACL) -> Role Permission Setting
  4. Create Designation from Left Menu->Human Resource->Designation
  5. Create Salary Grade from Left Menu->Payroll->Salary Garde
  6. Create Student Type from Left Menu->Student->Student Type
  7. Create Discount from Left Menu->Accounting->Discount
  8. Create Employee with Role Admin for the Specific School from Left Menu->Human Resource->Employee
  9. After create employee with admin role then set Role Permission for the Admin of the specific school
  10. Then School Admin can do every thing for his/her school

Step by Step Data Input Process:-

For smooth use of the software please follow the following data input process. These process will makes your software organized to data manipulation. You must follow these data input sequence.

  1. Login to System using default super admin user credentials: Email: superadmin Password: 123456
  2. Set your Brand General/Global Setting from Left Menu->Administrator->General Setting
  3. Create School as per your need/requirements. Left Menu->Administrator->Manage School
  4. Create Academic Year as per your school requirements. Left Menu->Administrator->Academic Year
  5. Manage/Create SMS Setting data for sending SMS. Left Menu->Administrator->SMS Setting {For Super Admin}
  6. Manage/CreateSMS Setting data for sending SMS. Left Menu->Setting->SMS Setting {Except Super Admin}
  7. Manage/Create SMS Setting data for manage online payment. Left Menu->Setting->Payment Setting {Except Super Admin}
  8. Manage/CreateEmail Setting data for sending Email. Left Menu->Setting->Email Setting {Except Super Admin}
  9. Manage/Create SMS Setting data for sending SMS. Left Menu->Administrator->SMS Setting {For Super Admin}
  10. Manage/Create Payment Setting data for manage online payment. Left Menu->Administrator->Payment Setting {For Super Admin}
  11. Manage/Create Email Setting data for sending Email. Left Menu->Administrator->Email Setting {For Super Admin}
  12. Create/Input Designation data for Manage Employee. Left Menu->Human Resource->Designation
  13. Create/Input Salary Grade data for Manage Payroll/Employee. Left Menu->Payroll->Salary Grade
  14. Create/Input Employees data as per your School requirements and for create school admin. Left Menu->Human Resources->Employees
  15. Update each User Type Role Permission as per your School requirements. Left Menu->Administrator->Role Permission->Role Permission Setting
  16. Create/Input Teachers data as per your School requirements. Left Menu->Teacher
  17. Create/Input Class data as per your School requirements. Left Menu->Class
  18. Create/Input Section data as per your Class & School requirements. Left Menu->Section->Class
  19. Create/Input Subject data as per your Class & School requirements. Left Menu->Subject->Class
  20. Create/Input Syllabus data as per your Class & School requirements. Left Menu->Syllabus->Class
  21. Create/Input Class Routine data as per your Class & School requirements. Left Menu->Class Routine->Class
  22. Create/Input Student Type data for create student. Left Menu->Student->Student Type
  23. Create/Input Discount data as per your School requirements. Left Menu->Accounting->Discount
  24. Create/Input Guardian data before input Student data. Left Menu->Guardian
  25. Create/Input Student data. Left Menu->Student->Student Class
  26. Manage Student/Teacher/Employee Attendance. Left Menu->Attendance->Student Attendance/ Teacher Attendance/ Employee Attendance
  27. Create/Input Assignment data for student. Left Menu->Assignment {Class Wise}
  28. Create/Input Exam Grade data as per School requirements. Left Menu->Exam->Exam Grade
  29. Create/Input Exam Term data as per School requirements. Left Menu->Exam->Exam Term
  30. Create/Input Exam Schedule data. Left Menu->Exam Schedule->Class
  31. Create/Input Exam Suggestion data. Left Menu->Exam Suggestion->Class
  32. Create/Input Exam Attendance data. Left Menu->Exam Attendance
  33. Manage Exam Mark data. Left Menu->Exam Mark->Manage Mark
  34. Manage Exam Term Result data. Left Menu->Exam Mark->Exam Term Result
  35. Manage Exam Final Result data. Left Menu->Exam Mark->Exam Final Result
  36. Manage Exam Merit List data. Left Menu->Exam Mark->Exam Merit List
  37. Manage Exam Mark Sheet data. Left Menu->Exam Mark->Student Mark Sheet
  38. Manage Student Result Card data. Left Menu->Exam Mark->Student Result Card
  39. Process Student Promotion to the next class after process all above data. Left Menu->Promotion
